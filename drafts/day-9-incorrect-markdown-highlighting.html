eBooks Library functions

I have a growing library of more than 350 digital ebooks. I bought most of them from Amazon (particulary from its Kindle Deal of the Day page, which has some excellent choices among the dross)

A short aside on nomenclature: ebook, e-book, eBook or e-Book? It seems there is no standard form for the term to describe a digital book. In this article I will use the simplest form ebook, whic aligns with email.

Calibre

Calibre is a powerful open source ebook manager. It provides the ability to read, stoire and convert ebooks in many formats.

I had most of my ebooks in Calibre, but were missing a few which were scattered elsewhere on my laptop's hard disk. I also wanted to check my Calibre library against my Amazon purchase history, but didn't want to do this manually.

I therefore wrote some Python code to extract the details of my Calibre library allowing me to easily manipulate and search the data.

Interestingly, Calibre is developed in Python, and actually includes an entire embedded copy of the Python interpreter. To access the Calibre database, I had to call my program with Calibre in debug mode rather than directly with Python:

calibre-debug <python-script>

The embedded interpreter had access to the standard Python libraries, but not to any other libraries I had installed wih pip. This meant that my program was limited to using standard Python library functions.

A further complexity was that the Python interpreter embedded in Calibre, at least as of Calibre version 4.15 in May 2020, is Python 2, specifically Python 2.7.23.

This meant I couldn't use any Python 3 code, which clashes with most of my other Python programs.

For these two reasons, I decided to create a standalone program to extract the Calibre database and save it to disk. I will leave the searching and comparison with Amazon to a separate Python 3 program.

First, I will import Calibre and also the Python pretty printer module pprint.

import calibre.library
import pprint

def get_calibre_books():
    """Get the list of books and authors from my Calibre eBook library."""
    # First open the Calibre library database and get a list of the book IDs
    calibre_db = calibre.library.db(
        ebooksconf.CALIBRE_LIBRARY_LOCATION
    ).new_api
    book_ids = calibre_db.all_book_ids()
    print("Got {} book IDs from Calibre database".format(len(book_ids)))

    # Create a dictionary with the book contents
    # The dictionary key is the book title
    # The dictionary value is a list of zero or more authors for the book
    books = {}
    for book in book_ids:
        title = calibre_db.field_for("title", book)
        if title in books:
            print("Duplicate book found for title {}".format(title))
        else:
            # The authors field contains a list of zero or more author names
            authors = calibre_db.field_for("authors", book)
            books[title] = authors
    return books

I can now wrap this in a main ()function 

if __name__ == "__main__":
    calibre_books = get_calibre_books()
    pprint.pprint(calibre_books)
